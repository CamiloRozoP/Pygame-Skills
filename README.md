# Pygame-Project
> Game developed using Python 3.4 and Pygame 1.93-3.4

# Requisitos:
* Tener Python 
* Tener Pygame

# Instrucciones
* Ejecuta el archivo game.py
* Enter para seleccionar jugar
* Usa W A S D para moverte, SPACE para disparar.
* SHIFT para tirar bombas, and P para poner pausa.
* Disfrutar
