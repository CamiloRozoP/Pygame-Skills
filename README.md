# Pygame-Project
> Game developed using Python 3.4 and Pygame 1.93-3.4

# Requisitos:
* Tener Python 
* Tener Pygame

# Instrucciones:
* Ejecuta el archivo dbz.py
* Up: arriba
* Down: abajo
* Left: izquierda
* Right: derecha
* S: disparar
* A: transformarse
* R: reiniciar
* Space: pausar
* Disfrutar
